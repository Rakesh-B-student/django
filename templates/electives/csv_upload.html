Column Required Example

Courses CSV

Column Required Example

title Yes Machine Learning

department_code Yes CSE
category Yes professional
level Yes ug
credits Yes 3
total_seats Yes 60
for_semesters Yes 5,6

Admin Users CSV

Column Required Example
username Yes admin
email Yes admin@sjbit.edu.in
first_name Yes Admin
last_name Yes User
password Yes admin123
Tips:
  • Save CSV files with UTF-8 encoding
  • Use comma (,) as the delimiter
  • Include column headers in the first row
  • Category must be: professional, open, or ability
  • Level must be: ug or pg
  • for_semesters format: "5,6" (comma-separated, no spaces)
